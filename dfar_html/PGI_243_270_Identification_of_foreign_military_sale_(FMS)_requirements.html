PGI 243.270 Identification of foreign military sale (FMS) requirements

When adding FMS requirements using a contract modification—  
  (1)  Clearly indicate “FMS Requirement” on the front of the modification; and  
  (2)  Refer to each FMS case identifier code by line/subline item number within the 
modification, e.g., FMS Case Identifier GY -D-DCA.
