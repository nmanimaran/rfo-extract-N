PGI 201.201-70 Numbering

(a)  Text that supplements the FAR uses the same numbering as its FAR counterpart, 
with the number 70 and up added to the FAR part, subpart, or section number .  When the 
text exceeds one paragraph, the paragraphs use the FAR numbering sequence, which is (a)(1)(i)(A)( 1)(i).  For example,  DFARS text supplementing FAR 1.101 would be numbered
